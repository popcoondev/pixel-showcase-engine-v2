# HD-2D 表現の分析と本エンジンへのマッピング

スクウェア・エニックスの「HD-2D」(OCTOPATH TRAVELER で確立、オクトパストラベラーII / トライアングルストラテジー / ドラクエ3リメイク等で進化)の映像仕様を分析し、Pixel Showcase Engine のパラメータに対応付ける。

## HD-2D を構成する要素

公開されているインタビュー・講演(CEDEC / 4Gamer / Unreal Engine 公式)から整理すると、HD-2D は UE4 上で次の要素を重ねたものと言える。

1. **ドット絵スプライト × 3D 環境**
   キャラはポイントサンプリング(ニアレスト)のドット絵を板ポリで配置し、背景は実3Dジオメトリ。ドットの色を保ったまま3Dのライティングを受ける。
2. **浅い被写界深度(ティルトシフト風)**
   画面の手前と奥を大きくボカし「箱庭 / ジオラマ / ミニチュア」感を作る。初期プロトタイプはパンフォーカス気味で「旧世代っぽい」と言われ、DOF を強くする方向に修正されたという経緯が語られている。HD-2D の最重要要素。
3. **ブルーム(発光の滲み)**
   ランタン・魔法・水面など高輝度部分を強めに滲ませ、ドット絵に「光っている」情報を足す。
4. **実影 + 暖色キーライト / 寒色環境光**
   点光源・平行光がスプライトにも実影を落とす。光は暖色、陰と環境は青寄りでコントラストを作る。
5. **距離フォグ(空気遠近)**
   奥を青暗く沈めて密度感と奥行きを出す。ボリュームライト・ゴッドレイ併用。
6. **周辺減光(ビネット)**
   強めのビネットで視線を中央に集め、ジオラマの「覗き込み」感を補強。
7. **俯瞰の固定構図・望遠寄りの圧縮**
   やや上から見下ろす固定カメラ。望遠寄りで奥行きを圧縮するとミニチュア感が増す。

## 本エンジンの「HD-2D Look」プリセット対応表

| HD-2D 要素 | 本エンジンのパラメータ | プリセット値 |
| --- | --- | --- |
| ドット絵の保持 | Material「ドット維持」(NearestFilter) | 既定で ON(変更しない) |
| 浅い DOF | DOF 有効 / 絞り / 焦点距離 | ON / f/1.8 / 50mm |
| ブルーム | Bloom 強さ | 1.3 |
| ビネット | 周辺減光 濃さ | 0.8 |
| 距離フォグ | Fog 色 / 開始 / 終了 | #141c33 / 12m / 50m |
| 寒色環境光 | Ambient 色 / 強さ | #7c8ec4 / 0.35 |
| 暖色キーライト | 先頭の Directional Light | #ffd9a8 / 3.5 / 影 ON(無ければ追加) |
| 沈んだ背景 | 背景色 / 地面色 | #0c0f1a / #171b26 |
| ややハイキー | 露出 | 1.15 |
| 俯瞰・固定構図 | カメラ位置(ユーザーが決める) | プリセットでは変更しない |

プリセットは Scene タブの「HD-2D風に調整」ボタンで適用する。1回の Undo (`Cmd/Ctrl+Z`) で適用前に戻せる。フォーカスモードや構図はユーザーの意図を尊重して変更しない。

## 演出エフェクト (FX タブ)

HD-2D 定番の演出は FX タブで「配置できるエフェクト」として管理する。ライトと同様に追加・選択・ギズモ移動・削除ができ、エフェクトごとに **色 / 粒の量 / 速さ / 大きさ / 広がり** をスライダーで調整できる。

- **キラキラの霧**: 細かい青白い粒が明滅しながらゆっくり沈む(加算ブレンド)
- **光の粒**: 蛍のような暖色の光球がゆっくり昇る。DOF と組み合わせると玉ボケになる
- **ダスト**: 空気中の塵。控えめな通常ブレンドで漂う
- **炎**: 根元から立ち昇り、明るく縮みながら消える
- **飛沫**: 噴水状に飛び散り重力で落ちる
- **電気**: 球状の範囲を高速で飛び移りながら明滅するスパーク
- **雨**: 縦ストリークの雨粒が降り注ぐ(わずかに流される)
- **風**: 淡い粒が波打ちながら水平に流れる。突風のような濃淡の揺らぎ付き

いずれも Scene JSON に保存され、Viewer でも再生される。実装はシェーダーポイント(粒ごとのサイズ・透明度制御)で、数百粒程度なら負荷はほぼない。
